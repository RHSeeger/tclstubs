NAME

loadStubs - Load in stub definitions

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    EXAMPLES
    SEE ALSO
    KEYWORDS

SYNOPSIS

::stubs::loadStubs module or file name

DESCRIPTION

The loadStubs command is used to load in stub definitions stored in a file. It can take either a module name, or an absolute file name. When given a module name, it searches several directories for a file by the name of "stubs.<module>.tcl".

The list of directories searched for modules is stored in ::stubs::libpaths, which is originally set the location of the stubs tcl file.

EXAMPLES

% set fd [open [file join $::stubs::libpaths stubs.testmodule.tcl] w]
% puts $fd {
stubs::define stub proc testStub {} { return testWorked }
}
% close $fd
% stubs::loadStubs testmodule
% stubs::stub defined testStub
% testStub
testWorked


% set fd [open [file join /tmp stubs.testmodule.tcl] w]
% puts $fd {
stubs::define stub proc testStub {} { return testWorked }
}
% close $fd
% stubs::loadStubs /tmp/stubs.testmodule.tcl
% stubs::stub defined testStub
% testStub
testWorked
