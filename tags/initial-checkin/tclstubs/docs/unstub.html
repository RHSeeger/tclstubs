NAME

unstub - Return stubbed commands back to their orignal definitions

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    SEE ALSO
    KEYWORDS

SYNOPSIS

::stubs::unstub ?commandName?

DESCRIPTION

The unstub command is used to return a command previously defined/redefined via stub back to its original definition. Given a command name, it returns the named command back to the definition it had (none, if it did not exist) before it was stubbed. If the command name is omitted, all commands that are currently stubbed are unstubbed.
