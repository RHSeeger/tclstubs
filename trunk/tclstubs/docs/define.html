NAME

define - Define a stub (or group of stubs) that can be used later

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    SEE ALSO
    KEYWORDS

SYNOPSIS

::stubs::define stub type name ?value?
::stubs::define group name script

DESCRIPTION

The define command is used to define a stub, or group of stubs, that can then be used later via the stub defined functionality.

::stubs::define stub type name ?value?
Defines a stub for a command, that can later be put into place using stub defined. It is generally used for simple commands that will be stubbed out multiple times. 

% ::stubs::define stub ensemble clock seconds { return 0 }
% clock format 1103523988 -format %D -gmt 1
07/11/04
% ::stubs::stub defined clock
% clock format 1103523988 -format %D -gmt 1
01/01/70



::stubs::define group name script
Defines a stub group and the script to run when it is stubbed via stub group. 

% ::stubs::define group mygroup {
    stubs::stub proc firstCommand {} { return first }
    stubs::stub proc secondCommand {} { return [incr ::mygroupcounter] }
    set mygroupcounter 0
}
% ::stubs::stub group mygroup
% firstCommand
first
% secondCommand
1
% secondCommand
2
